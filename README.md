И уже там пройдёт вставка или проверка на то куда вставить дальше.

#### Функция дублирования: treeMap посмотрит на корень, увидит там число и отправит его в dublicateNumber где число временно приводится к строке, склеивается само с собой, а затем снова преобразуется в целое число. treeMap получит число обратно и пойдёт по веткам дерева чтобы и там дублировать числа.

## Задание 2



### Задача 1



### Текст задачи



#### Выяснить, содержится ли в дереве элемент с заданным значением.



### Алгоритм решения



#### В функции main происходит: ввод кол-ва элементов в дереве, вызов создания дерева(generateRandom) сo случайным кол-вом элементов в нём, запрос ввода элемента для поиска, вызов функции treeFold с параметрами для поиска элементов, изначальным флагом false, самим деревом.

#### В функции создания дерева (generateRandom) происходит: если кол-во всех элементов != 0, то генерируется случайное число, уменьшение кол-ва элементов, вызов функции вставки(insert) со случайным значением.

#### В функции вставки (insert) происходит проверка: если встречается пустое поле, то Создаётся head с текущим числом и ветками null(как null в C++). Если встретилась голова с текущим значением и ссылками на ветви, то проверяется: если новое число меньше значения в узле, оно рекурсивно отправляется влево, если больше - то рекурсивно вправо. И уже там пройдёт вставка или проверка на то куда вставить дальше.

#### Функция treeFold - это жадный обход. В ней происходит заход сначала в корень, а затем сначала в левую ветку, а потом в правую. Когда программа найдёт число, которое ищем, аккумулятор поменяет своё значение на true. Дальше этот флаг передастся по цепочке через все остальные узлы. 

#### Отличие treeMap от treeFold в том, что:
##### treeMap - функция трансформации. Заходит в каждый узел дерева, изменяет его значение с помощью переданной функции, но полностью сохраняет исходную структуру дерева. На выходе новое дерево такого же размера.
##### treeFold - функция свёртки. Она обойдёт дерево и сожмёт его значения до одного значения. В этом коде используется флаг и переносимый параметр - аккумулятор acc, который передаётся из узла в узел.
