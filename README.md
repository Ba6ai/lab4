### Алгоритм решения


#### В функции main происходит: ввод кол-ва элементов в дереве, вызов создания дерева(readTree) сo случайным кол-вом элементов в нём, вызов функции для дублирования элементов в нём.

#### В функции создания дерева (readTree) происходит: если кол-во всех элементов != 0, то генерируется случайное число, уменьшение кол-ва элементов, вызов функции вставки(insert) со случайным значением.

#### В функции вставки (insert) происходит проверка: если встречается пустое поле, то Создаётся head с текущим числом и ветками null(как null в C++). Если встретилась голова с текущим значением и ссылками на ветви, то проверяется: если новое число меньше значения в узле, оно рекурсивно отправляется влево, если больше - то рекурсивно вправо. И уже там пройдёт вставка или проверка на то куда вставить дальше.

#### Функция дублирования: treeMap посмотрит на корень, увидит там число и отправит его в dublicateNumber где число временно приводится к строке, склеивается само с собой, а затем снова преобразуется в целое число. treeMap получит число обратно и пойдёт по веткам дерева чтобы и там дублировать числа.
